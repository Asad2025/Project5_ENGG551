# ENGO651 / S51 - Lab 5

## Overview
This project demonstrates the use of MQTT over WebSockets in a JavaScript-based web application, along with integration of the browser Geolocation API.

## Features
- Connect to MQTT broker using WebSockets
- Subscribe to a topic and receive real-time messages
- Publish messages to a topic
- Retrieve and display user geolocation
- Send geolocation data via MQTT

## Live Demo
https://asad2025.github.io/Project5_ENGG551/

## Technologies Used
- JavaScript
- MQTT.js
- HTML/CSS
- Mosquitto public broker

## How to Use
1. Click **Connect** to connect to the MQTT broker
2. Click **Subscribe** to listen to a topic
3. Enter a message and click **Publish**
4. Click **Send My Location** to share geolocation
